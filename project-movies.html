تطبيق أفلام ومسلسلات تم تطويره باستخدام Flutter، يتيح للمستخدمين استعراض الأفلام والمسلسلات الشائعة، مشاهدة تفاصيل العمل، تقييم المحتوى، وإضافة العناصر إلى المفضلة. يعتمد التطبيق على استهلاك API خارجي لجلب البيانات بشكل مباشر مع واجهة مستخدم حديثة ومتجاوبة.
