## 🚀 كيفية الاستخدام

### 1. التثبيت والتشغيل
```bash
# استنساخ المشروع أو تحميل الملفات
cd portfolio-enhanced

# فتح الملف في المتصفح
# يمكنك فتح index.html مباشرة في المتصفح
```

### 2. تخصيص المحتوى
قم بتعديل المحتوى في `index.html`:
- غير اسمك والوصف
- أضف صورك الخاصة في مجلد `images/`
- حدّث بيانات التواصل (البريد والهاتف)
- أضف روابط وسائل التواصل الاجتماعية

### 3. تخصيص التصميم
عدّل الألوان والخطوط في `style.css`:
```css
:root {
  --primary: #5E4BFA;        /* اللون الأساسي */
  --primary-dark: #4b3bd2;   /* اللون الأساسي الداكن */
  --secondary: #FF6B6B;      /* اللون الثانوي */
  --accent: #4ECDC4;         /* لون التأكيد */
}
```

### 4. إضافة مشاريع جديدة
أضف بطاقة مشروع جديدة في `index.html`:
```html
<div class="project-card" data-category="flutter">
  <div class="project-image">
    <img src="images/your-project.jpg" alt="اسم المشروع">
    <div class="project-overlay">
      <a href="#" class="project-link" title="عرض المشروع">
        <i class="fas fa-external-link-alt"></i>
      </a>
    </div>
  </div>
  <div class="project-info">
    <h3>اسم المشروع</h3>
    <p class="project-category">نوع المشروع</p>
    <p class="project-description">وصف المشروع</p>
    <div class="project-tech">
      <span>التقنية 1</span>
      <span>التقنية 2</span>
    </div>
  </div>
</div>
```

## 🎨 التخصيص المتقدم

### تغيير الألوان
عدّل متغيرات CSS في `:root`:
```css
--primary: #your-color;
--secondary: #your-color;
--accent: #your-color;
```

### إضافة رسوم متحركة جديدة
أضف keyframes جديدة في `style.css`:
```css
@keyframes yourAnimation {
  from { /* البداية */ }
  to { /* النهاية */ }
}
```

### تعديل الخطوط
غيّر الخطوط في `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont&display=swap" rel="stylesheet">
```
