نظام الجرد الذكي بالذكاء الاصطناعي

العودة للمشاريع

نبذة عن المشروع

نظام الجرد الذكي هو مشروع متكامل لإدارة المخزون باستخدام تقنيات الذكاء الاصطناعي ورؤية الحاسوب. يتيح النظام للمستخدم التقاط صورة للمنتجات أو رفعها من المعرض ليتم تحليلها تلقائياً بواسطة نموذج YOLOv8 للتعرف على المنتجات وحساب الكميات بدقة عالية.

التقنيات المستخدمة

Flutter Dart ASP.NET Blazor Python Flask YOLOv8 SQL Server JWT Auth

التفاصيل الكاملة

يعتمد المشروع على بنية متكاملة تتكون من تطبيق Flutter، وخدمة ذكاء اصطناعي مطورة باستخدام Python & Flask، وواجهة ASP.NET Web API لمعالجة البيانات والتواصل بين الأنظمة، بالإضافة إلى لوحة تحكم احترافية مطورة باستخدام Blazor لإدارة المنتجات والمخزون والتقارير والمستخدمين.

أهم المميزات

الذكاء الاصطناعي

إدارة المخزون

التقارير

إدارة المستخدمين

إدارة المستخدمين والصلاحيات والأدوار المختلفة

الأمان
